Add courier and tracking number to WooCommerce orders and track them on TRACK718.

== Description ==
TRACK718 is a one-stop global logistics trajectory query platform. It connects cross-border e-commerce sellers, cross-border e-commerce buyers, third-party international logistics companies, international freight forwarders and delivery carriers in series. Personalized trajectory customization service is provided for e-commerce sellers and logistics service providers who need comprehensive trajectory information management, so as to realize monitoring on data lines, improve operation effect, solve the problem of knowing nothing about the transportation of goods in the process of cross-border transportation, and realize efficient visual tracking.

= Free Installation =
* One click installation TRACK718 plugin to your admin panel.

= Add Tracking Number AND Courier =
* Auto-tracking with 1000+ global carriers supported
* Install the TRACK718 plugin to automatically add tracking number and courier fields to your WooCommerce store's order.
* TRACK718 no actual fulfill your order. if you want to do this. Please contact the actual courier.

= Show Shipment Info To Your Customers =
* After you install TRACK718 plugin, your customers are able to track the tracking number in in your WooCommerce store.

== Installation ==

* This plugin requires at least PHP 5.2.4
* This plugin requires at least WooCommerce 3.4
